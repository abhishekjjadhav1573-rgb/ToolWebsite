Create a modern **single-page tools website** called **"Smart Utility Hub"** using **HTML, CSS, and JavaScript**.

The website must have a **clean, minimal, professional design** similar to modern SaaS websites.

IMPORTANT DESIGN REQUIREMENTS:

• The website must be **single page (SPA style)**.
• All tools should appear **on the same page dynamically when selected** (do not reload the page).
• Use **JavaScript to show/hide tools** when users click a tool from the sidebar or menu.
• The layout should be **fast, lightweight, and SEO friendly**.
• The design should look **modern, clean, and professional**.

DESIGN STYLE:

• Minimal UI
• White background with soft shadows
• Rounded cards
• Smooth hover effects
• Professional typography
• Responsive mobile-first design
• Smooth transitions when switching tools

ADS DESIGN REQUIREMENTS:

The website will use **Google AdSense**, so include **dedicated ad spaces** that blend naturally with the layout.

Important rules:

• Ads must **not break the layout or visual design**
• Ads must **look integrated and balanced with the UI**
• Use **clean placeholder containers labeled "Ad Space"**
• Ads should appear inside **soft bordered containers or cards**
• Ad placement should **not interrupt tool usability**

Recommended ad placements:

• One **leaderboard ad below the header**
• One **sidebar ad block** below tool categories
• One **ad space between tool sections** in the main content
• One **footer ad block above the footer**

All ad containers must:

• Match the website card style
• Have proper spacing and margins
• Be responsive for mobile and desktop
• Avoid cluttering the interface
